You can deploy your apps via Github Repo, SSH, CI/CD Pipelines

### Create an empty app on Bunetic.com

### Step 1: Click on "Deploy New App"
![Deploy New App](../../../assets/create-empty-app/1.png)

### 2: Choose a name for your app
![Choose App Name](../../../assets/create-empty-app/2.png)

### 3: Choose a server
![Choose Server](../../../assets/create-empty-app/3.png)

### 4: Add SSH Key
![Add SSH Key](../../../assets/create-empty-app/4.png)

### 5: Wait a moment to finish set up
![Setup Progress](../../../assets/create-empty-app/5.png)

![Setup Complete](../../../assets/create-empty-app/6.png)

## Deploy via Github Repo

### 6: Choose Manage
![Choose Manage](../../../assets/create-empty-app/7.png)

### 7: Choose Deploy
![Choose Deploy](../../../assets/create-empty-app/8.png)

### 8: Connect Github
![Connect Github](../../../assets/create-empty-app/9.png)

### 9: Connect Repo
![Connect Repo](../../../assets/create-empty-app/10.png)

### 10: After connected, you can choose Manual Deploy or Auto Deploy
![Manual Deploy](../../../assets/create-empty-app/11.png)

![Auto Deploy](../../../assets/create-empty-app/12.png)

### 11: Done!
![Deployment Complete](../../../assets/create-empty-app/13.png)
